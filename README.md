# zcouncil skill

[![skills.sh](https://skills.sh/b/zcouncil/skill)](https://skills.sh/zcouncil/skill)

A public agent skill for using [zcouncil](https://zcouncil.com): multiple independent AI perspectives from one prompt.

## Install

With the Skills CLI:

```sh
npx skills add zcouncil/skill --skill zcouncil
```

List the skill without installing:

```sh
npx skills add zcouncil/skill --list
```

## What it teaches agents

- when to use zcouncil
- how to prompt a council neutrally
- how to use zcouncil through Pi, MCP, the CLI, or the TypeScript SDK
- how to avoid token/secret leakage
